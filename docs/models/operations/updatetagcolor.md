# UpdateTagColor

The color of the tag. If not provided, a random color will be used from the list: red, yellow, green, blue, purple, brown, gray.


## Values

| Name     | Value    |
| -------- | -------- |
| `RED`    | red      |
| `YELLOW` | yellow   |
| `GREEN`  | green    |
| `BLUE`   | blue     |
| `PURPLE` | purple   |
| `BROWN`  | brown    |
| `GRAY`   | gray     |
| `PINK`   | pink     |
