# Interval

The interval to retrieve analytics for. Takes precedence over start and end. If undefined, defaults to 24h.


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `TWENTY_FOURH`   | 24h              |
| `SEVEND`         | 7d               |
| `THIRTYD`        | 30d              |
| `NINETYD`        | 90d              |
| `YTD`            | ytd              |
| `ONEY`           | 1y               |
| `ALL`            | all              |
| `ALL_UNFILTERED` | all_unfiltered   |
