# SortBy

The field to sort the links by. The default is `createdAt`.

## Example Usage

```python
from dub.models.operations import SortBy

value = SortBy.CREATED_AT
```


## Values

| Name           | Value          |
| -------------- | -------------- |
| `CREATED_AT`   | createdAt      |
| `CLICKS`       | clicks         |
| `SALE_AMOUNT`  | saleAmount     |
| `LAST_CLICKED` | lastClicked    |
