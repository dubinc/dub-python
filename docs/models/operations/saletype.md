# SaleType

Filter sales by type: 'new' for first-time purchases, 'recurring' for repeat purchases. If undefined, returns both.

## Example Usage

```python
from dub.models.operations import SaleType

value = SaleType.NEW
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `NEW`       | new         |
| `RECURRING` | recurring   |
