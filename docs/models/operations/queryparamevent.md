# QueryParamEvent

The type of event to retrieve analytics for. Defaults to 'clicks'.

## Example Usage

```python
from dub.models.operations import QueryParamEvent

value = QueryParamEvent.CLICKS
```


## Values

| Name     | Value    |
| -------- | -------- |
| `CLICKS` | clicks   |
| `LEADS`  | leads    |
| `SALES`  | sales    |
