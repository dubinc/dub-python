# Event

The type of event to retrieve analytics for. Defaults to `clicks`.

## Example Usage

```python
from dub.models.operations import Event

value = Event.CLICKS
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `CLICKS`    | clicks      |
| `LEADS`     | leads       |
| `SALES`     | sales       |
| `COMPOSITE` | composite   |
