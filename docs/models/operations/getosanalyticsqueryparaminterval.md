# GetOSAnalyticsQueryParamInterval

The interval to retrieve analytics for. Takes precedence over start and end.


## Values

| Name           | Value          |
| -------------- | -------------- |
| `ONEH`         | 1h             |
| `TWENTY_FOURH` | 24h            |
| `SEVEND`       | 7d             |
| `THIRTYD`      | 30d            |
| `NINETYD`      | 90d            |
| `YTD`          | ytd            |
| `ONEY`         | 1y             |
| `ALL`          | all            |
