# QueryParamGroupBy

The parameter to group the analytics data points by. Defaults to 'count' if undefined.


## Values

| Name           | Value          |
| -------------- | -------------- |
| `COUNT`        | count          |
| `TIMESERIES`   | timeseries     |
| `CONTINENTS`   | continents     |
| `COUNTRIES`    | countries      |
| `CITIES`       | cities         |
| `DEVICES`      | devices        |
| `BROWSERS`     | browsers       |
| `OS`           | os             |
| `REFERERS`     | referers       |
| `REFERER_URLS` | referer_urls   |
| `TOP_LINKS`    | top_links      |
| `TOP_URLS`     | top_urls       |
| `TRIGGER`      | trigger        |
