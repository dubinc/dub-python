## Overview

### Available Operations

## create

Create a folder for the authenticated workspace.

### Example Usage

```python
from dub import Dub


with Dub(
    token="DUB_API_KEY",
) as d_client:

    res = d_client.folders.create()

    assert res is not None

    # Handle response
    print(res)
```
