# Partners
(*partners*)

## Overview

### Available Operations

* [create](#create) - Create a new partner

## create

Create a new partner for a program. If partner exists, automatically enrolls them.

### Example Usage

```python
from dub import Dub

with Dub(
    token="DUB_API_KEY",
) as dub:

    res = dub.partners.create()

    assert res is not None

    # Handle response
    print(res)

```
