# QRCodes
(*qr_codes*)

## Overview

## get

Retrieve a QR code for a link.

### Example Usage

```python
from dub import Dub

s = Dub(
    token="DUB_API_KEY",
)

res = s.qr_codes.get(request={
    "url": "https://brief-micronutrient.org",
})

if res is not None:
    # handle response
    pass

```
